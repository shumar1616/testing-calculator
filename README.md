# Calculator App 🧮

Ek simple aur beautiful calculator app jo modern UI ke saath bana hai.

## Features ✨

✅ Maqam Calculator  
✅ AC aur DEL buttons  
✅ Beautiful purple gradient UI  
✅ Keyboard support (numbers, operators, Enter, Backspace, Escape)  
✅ Responsive design (mobile aur desktop)  
✅ Error handling  
✅ Decimal number support  

## Files 📁

- `index.html` - Main HTML structure
- `style.css` - Styling aur responsive design
- `script.js` - Calculator logic aur keyboard events
- `README.md` - Documentation

## Keyboard Shortcuts ⌨️

| Key | Function |
|-----|----------|
| 0-9 | Numbers |
| + - * / | Operators |
| . | Decimal |
| Enter / = | Calculate |
| Backspace | Delete last |
| Escape | Clear all |

## Local Testing 🖥️

5. Browser mein calculator khul jayega!
